## 새 기능

### 1. 글로벌 디자인 시스템 (core/ui_theme.py)
- Pretendard 한국어 폰트 자동 로드
- 그린리모델링 컬러 팔레트 (딥 그린 + 라임 + 어스 톤)
- SVG 로고 (빌딩 + 잎 아이콘)
- 카드/배지/푸터 헬퍼 함수

### 2. 메인 랜딩 페이지 (홈)
- 사이드바에 "🏠 홈" 추가 (기본 선택)
- 히어로 영역 + 핵심 수치 (도담 검증 결과)
- 4개 모드 카드 (API 키 상태 자동 표시)
- 빠른 시작 가이드 + 기술 스택

### 3. Mode 3 BIM 진단 시각화 강화
- **총점 SVG 도넛 게이지** (등급 색상별)
- **진행바 with 그라데이션** (GR 요소 80점, 사업여건 20점)
- **11개 카드 그리드** — 상태 색상 + 비율 바
- **점수 분해 막대 차트** — SVG 기반
- **🏆 Top 3 보강 항목 카드** — 금/은/동 메달
- **누적 효과 SVG 차트** — 점수 vs 비용 듀얼 라인

### 4. Mode 1/2/4 헤더 일관화
- 모드별 영문 카테고리 라벨
- 큰 제목 + 부제 + 출처 안내

### 5. 사이드바 디자인
- SVG 로고
- 모드 선택 라디오 (호버 효과)
- 시스템 상태 점검 (Claude API + RAG 인덱스 + 단가DB)
- 프로젝트 정보 확장 가능
- 안전 안내 푸터

### 6. CSS 글로벌 스타일
- 메트릭 박스 그라데이션 + 호버 그림자
- 탭 헤더 활성화 시 컬러 강조
- Primary 버튼 그라데이션 + 호버 효과
- 진행바 그린 그라데이션
- 데이터프레임 라운드 모서리

## 디자인 컨셉

**테마**: 그린리모델링 — 친환경 + 데이터 분석
**컬러**: `#1B5E20` (딥그린) + `#76FF03` (라임) + `#5D4037` (어스) + 화이트 베이스
**무드**: 깔끔한 대시보드 (네이버/애플 풍)
**폰트**: Pretendard (한국어 최적화 무료 폰트)
